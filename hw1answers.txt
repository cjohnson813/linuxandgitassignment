Linus Commands:

- Entering Google Cloud VM: ssh cjohnson813@34.16.39.19
- Creating a new directory: mkdir <directory_name>
- Change to existing directory: cd <directory_name>

- ls: List files in currently selected directory
(-ltr): list one file per line, sort by newest, reverse order
- cd .. : Exit current directory and reference one directory up
- cat <html.file>: Check contents (code/txt) of the requested file
